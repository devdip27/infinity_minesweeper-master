## About The Project


Infinity Minesweeper is written using Flutter.

Include Simon Tatham's No Guess algorithm.

All UI using Neumorphic Design.

## Features

* No guessing algorithm
* Multi languages
* Statistics!
* Easy, Medium and Hard mode.
* Leaderboard
* Pro version
* Tutorial
